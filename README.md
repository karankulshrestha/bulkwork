
# BulkWork - Karan Kulshrestha

An App which help in encouraging the people to Design a Customized workout plan for a year, month or even for week with customized according to their routine. This app contains custom graphs for monitoring different muscles of the body along with a section for calculating protein and fat. User also read custom blogs for Bulking, Cutting, etc


## Tech Stack

**App:** Flutter, Firebase Auth, Firestore Database, Multiple Libraries, syncfusion Graphs.
